Flex box

he Flexbox Layout (Flexible Box) module (a W3C Candidate Recommendation as of October 2017) aims at providing a more efficient way to lay out, align and distribute space among items in a container, even when their size is unknown and/or dynamic (thus the word “flex”).

Flex container/parent

This defines a flex container; inline or block depending on the given value. It enables a flex context for all its direct children.\nThis establishes the main-axis, thus defining the direction flex items are placed in the flex container. Flexbox is (aside from optional wrapping) a single-direction layout concept. Think of flex items as primarily laying out either in horizontal rows or vertical columns.

Flex item/child

By default, flex items are laid out in the source order. However, the order property controls the order in which they appear in the flex container.
